## 项目说明

构建一个移动端 H5 网页（React + Tailwind），实现"输入文字 → 一键生成 3 种风格排版结果 → 复制"的工具。视觉为「治愈卡通 + 复古像素」融合，配色采用 **燕麦白 + 暖橘 + 雾紫**。完全前端、无后端、无登录。

> 说明：原需求是 uniapp 微信小程序，但 Lovable 仅支持 React Web。最终产物为移动端 H5 页面，可在手机浏览器或微信内打开使用。

---

## 设计系统

- **配色**（HSL，写入 index.css 设计 token）
  - 背景：燕麦白 `#FAF6EC`
  - 主色 / 强调：暖橘 `#F5A26B`
  - 次强调：雾紫 `#B9A7D6`
  - 文字主色：深棕灰 `#3F352E`
  - 卡片背景：纯白带轻微米色
  - 边框：低饱和米灰，2px 像素硬边
- **字体**：标题用像素风字体（Press Start 2P / VT323，Google Fonts 引入），正文用清爽无衬线（Noto Sans SC）
- **圆角**：12–16px 卡片圆角；像素装饰元素用直角
- **像素装饰**：8-bit 风小图标（星星、爱心、云朵、像素边框），SVG 内联，点缀不堆砌
- **动画**：按钮按下回弹、结果卡片淡入上滑、复制成功 toast

## 页面结构（单页）

### 1. 顶部 Header
- 居中像素风标题「文字排版机」
- 副标题小字「一键生成三种风格」
- 左右各一个像素小图标（星星 / 爱心）作点缀

### 2. 输入区
- 大尺寸 textarea（最少 6 行高），圆角 + 像素硬边框
- placeholder 示例文案
- 右下角字数统计
- 下方有「清空」次要按钮

### 3. 主操作按钮
- 像素风「✨ 一键排版 ✨」大按钮，暖橘底色，带像素阴影（box-shadow 偏移模拟 8-bit 凸起）
- 点击有按下位移反馈
- 输入为空时禁用

### 4. 结果展示区（生成后出现）
三张卡片纵向排列，每张包含：
- 卡片标题 + 风格小图标（像素徽章）
- 排版后文字预览（保留换行、等宽展示）
- 右上角「📋 复制」按钮，复制成功变成「✓ 已复制」并 1.5 秒后恢复

**三种风格生成规则**：

| 风格 | 处理逻辑 |
|---|---|
| ① 简约清爽风 | 按句号/问号/感叹号切句，每句独占一行；合并多余空格；段落间空一行 |
| ② 文艺治愈风 | 顶部 `─── ✿ ───` 装饰线；每段前加 `「` 后加 `」`；段间用 `· · ·` 分隔；末尾加 `— 致每个温柔的你 —` 签名线 |
| ③ 潮流个性风 | 句首随机插入 emoji（✨🔥💫🌙☁️ 池）；自动从文本提取 2–3 个关键词加 `#标签#`；段落用 `▍` 起头 |

### 5. 底部
- 极简版权 / 提示小字「在微信中打开可直接复制分享」

### 6. 全局交互
- 复制使用 `navigator.clipboard`，失败降级 textarea+execCommand（兼容微信 webview）
- 复制成功用 sonner toast 提示
- 全页响应式，max-width 容器（约 480px）居中，留白舒适
- 移动端首选，桌面端居中显示手机宽度
